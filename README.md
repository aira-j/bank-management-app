# 🏦 Bank Management System

A beginner-friendly **Python-based Bank Management System** built using Object-Oriented Programming (OOP).  
This is an application to simulate basic banking operations like account creation, login, deposit, withdrawal, and balance check.

## 🚀 Features
- Create a new account
- Secure login system
- Deposit money
- Withdraw money
- Check account balance
- Password verification
